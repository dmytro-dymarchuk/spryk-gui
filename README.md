# SprykGui Module

GUI for [Spryk](https://github.com/spryker-sdk/spryk/) tool. The Spryk tool can be used standalone, for optimized usage in Spryker this module offers a GUI in Zed.

## Installation

```
composer require --dev spryker-sdk/spryk-gui
```
